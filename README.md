# DHDE Prognostics Model

This project implements a physics-based predictive maintenance model called the Dynamic Health State Differential Equation (DHDE) framework for rolling bearing failure prediction.

## Overview

The model uses real vibration sensor data from the NASA Prognostics Data Repository to:

- Estimate equipment health state
- Track degradation dynamics
- Predict Remaining Useful Life (RUL)
- Perform real-time adaptive failure prediction

## Key Features

- Differential equation–based degradation modeling
- Adaptive parameter estimation using sliding window regression
- Real-time failure prediction capability
- Validated on NASA bearing degradation dataset
- Achieved 2–7% failure prediction error near end-of-life

## Technologies Used

- MATLAB
- Signal processing
- Reliability engineering principles
- Prognostics and Health Management (PHM)

## Applications

- Rotating equipment reliability
- Predictive maintenance
- Oil & Gas rotating machinery
- Industrial asset integrity monitoring
