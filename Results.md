# Results

This folder contains output plots from the DHDE prognostic model.

Key outputs include:

- Adaptive degradation exponent estimation
- Real-time Remaining Useful Life (RUL) prediction
- Prediction error convergence

These results demonstrate the model's ability to track degradation and predict failure accurately.
